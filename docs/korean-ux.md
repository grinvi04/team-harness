# 한국어 UI/UX 표준 (한국 서비스·한국어 화면 전용)

> 한국 사용자를 위한 화면 문구·용어·폼 포맷의 **단일 출처**. AI가 작업 시 영어 직역체·서구 UX 기본값으로
> 폴백하는 것을 막는다. AI 작동에 직접 영향을 주는 룰은 `templates/rules/korean-ux.md`(repo 복사본),
> advisory 점검은 `/qa` Agent C. **영어 UI 서비스에는 적용하지 않는다.**

## 1. 마이크로카피 톤 — 해요체 기본

업무 시스템·일반 서비스 = **해요체**(~해요/~이에요/~할까요?). 공식 공지·약관은 합쇼체(~합니다) 허용.
버튼은 **동사 단독**, 영어 버튼명(Save·Cancel·Delete) 금지.

| 상황 | ❌ 직역체 | ✅ 한국식 |
|---|---|---|
| 저장 확인 다이얼로그 | "저장하시겠습니까?" | "저장할까요?" |
| 삭제 경고 | "이 항목을 삭제하시겠습니까?" | "정말 삭제하시겠어요? 되돌릴 수 없어요." |
| 저장 성공 토스트 | "저장 되었습니다" | "저장했어요" |
| 삭제 성공 | "삭제 되었습니다" | "삭제했어요" |
| 이메일 중복 에러 | "이미 존재하는 이메일입니다" | "이미 사용 중인 이메일이에요" |
| 필수 필드 에러 | "이 필드는 필수입니다" | "필수 항목이에요" |
| 빈 상태 | "데이터가 없습니다" | "아직 내역이 없어요" |
| 로딩 중 | "로딩 중..." | "불러오는 중..." |
| 검색 결과 없음 | "결과가 없습니다" | "검색 결과가 없어요" |

- 확인 버튼: 동사 단독("저장"·"삭제"·"확인"). 취소 버튼: "취소".
- 검색창 placeholder: `검색어를 입력하세요` (not `Search...`).

## 2. 정착 UI 용어집 — 직역 금지

| ❌ 직역체 | ✅ 정착어 | 비고 |
|---|---|---|
| 내 페이지 / 프로필 페이지 | **마이페이지** | 한국 웹 사실상 표준 |
| 쇼핑 카트 / 카트 | **장바구니** | 커머스 절대 기준 |
| 연락하기 / 컨택 | **문의하기** | |
| 알림(공지 의미) / 뉴스 | **공지사항** | 게시판 메뉴 |
| 서비스 약관 | **이용약관** | |
| 프라이버시 정책 | **개인정보처리방침** | 법령 용어 |
| 가입 / 사인업 | **회원가입** | |
| 사인인 / 로그인 | **로그인** | (영어 정착어로 허용) |
| 사인아웃 | **로그아웃** | |
| 사용자 이름 / 유저네임 | **아이디** | |
| 패스워드 | **비밀번호** | |
| 서치 / 검색하기 | **검색** | |

## 3. 폼·데이터 포맷

- **이름**: 단일 필드(성+이름 분리 없음). placeholder `홍길동`.
- **휴대폰**: `010-XXXX-XXXX`(하이픈 자동 삽입, 010 prefix). 일반전화는 지역번호 가변(`02-XXX-XXXX`).
- **사업자등록번호**: `XXX-XX-XXXXX` 10자리(하이픈 자동).
- **주소**: 우편번호 검색(카카오/다음 주소 API) → 도로명주소 자동완성 → 상세주소. 2필드, 우편번호 5자리.
- **원화 금액**: `1,234,567원` 또는 `₩1,234,567`. **소수점 금지**(정수). KPI 축약 `1억 2,345만`. tabular 폰트.
- **날짜**: 화면 표시 `2026.01.15`(점 구분) 또는 `2026년 1월 15일`. **ISO 8601(YYYY-MM-DD)은 API 전송용**이지 화면 표시용 아님.
- **시간**: 업무 시스템 24시간제(`14:30`), 사용자 대면은 `오후 2:30` 허용.
- **퍼센트**: `12.5%`(숫자+% 붙여쓰기).

## 4. 한국 UI 관행

- 다이얼로그 버튼 순서: **취소(왼쪽·secondary) → 확인(오른쪽·primary)**. 삭제 경고는 취소(왼쪽) → 삭제(오른쪽·destructive).
- 토스트/스낵바: 하단 중앙 기본.
- 테이블: 헤더·텍스트 좌측 정렬, **숫자·금액 우측 정렬**.
- 사이드바 없을 때 상단 탭 메뉴는 한국 업무 시스템 표준 UX.
